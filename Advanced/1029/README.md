## 分析

由于给的为两个有序sequence，所以我们不用进行排序。

计n, m分别为两个sequence的大小，那么median在两个sequence合成的第 (n + m - 1) / 2位子。// 由于0下标记录，所以是这样的结果。

那么就需要把两个队列的前面数字弹出，显然使用queue的数据结构。
